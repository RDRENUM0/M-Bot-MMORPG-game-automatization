# ⚔️ Automation Bot (WIP)
> **Status:** 🚧 Work in Progress

Zautomatyzowany asystent do gry MMORPG oparty na technologii **Selenium WebDriver**. Projekt stworzony w celach edukacyjnych, demonstrujący możliwości automatyzacji przeglądarki i symulacji zachowań użytkownika.

---

## 📝 O projekcie
Bot ma na celu odciążenie gracza w powtarzalnych czynnościach (grindzie) oraz optymalizację poruszania się po świecie gry. Skupia się na precyzyjnym odczytywaniu stanu gry z DOM oraz szybkim reagowaniu na zdarzenia.

**Obecnie rozwijane funkcjonalności:**
* 🗺️ **Auto-walking:** Inteligentne poruszanie się między mapami.
* ⚔️ **Auto-combat:** Automatyczne podejmowanie walki z potworami (e-elity, herosi).
* 🎒 **Loot Manager:** Automatyczne zbieranie przedmiotów i zarządzanie ekwipunkiem.
* 🧪 **Healer:** Monitorowanie poziomu HP i automatyczne używanie mikstur.
i wiele więcej

## 🛠️ Stack Technologiczny
* **Silnik:** [Selenium WebDriver](https://www.selenium.dev/)
* **Język:** Python
* **Przeglądarka:** Chrome (ChromeDriver)
